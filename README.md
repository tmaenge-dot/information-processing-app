# 📄 Information Processing & Typewriting Tutor App

A comprehensive **Information Processing & Typewriting Tutor Application** designed for the **National Certificate in Secretarial Studies**. This modern web application provides structured learning, speed development, and assessment tools aligned with the official syllabus.

## 🎯 Project Overview

### Target Users
- **Educational Institutions**: Colleges offering National Certificate in Secretarial Studies
- **Individual Learners**: Students pursuing information processing and typewriting skills
- **Professional Development**: Working professionals improving typing skills

### Syllabus Alignment
The app follows the official Information Processing syllabus structure:

**Part I - Information Processing I Part I (Target: 25 WPM)**
- **A. Introduction**: Computer care, hardware/software usage, system operations
- **B. Keyboard Mastery**: Keyboarding techniques, corrections, proofreading
- **C. Elementary Skills**: Document formatting, business documents, basic layouts

**Part II - Information Processing I Part II (Target: 35 WPM)**
- **D. Intermediate Skills**: Advanced formatting, tables, manuscripts, forms

## 🛠 Installation & Setup

### Prerequisites
- Node.js 18+ 
- npm or yarn package manager
- Modern web browser

### Local Development
```bash
# Clone the repository
git clone [repository-url]

# Navigate to project directory
cd information-processing-app

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Development Server
- **Local**: http://localhost:3002
- **Network**: Available on local network for testing
